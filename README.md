# NoteTaker — WPF заметки с SQLite

Простой и удобный дневник заметок для рабочего стола Windows.

## ✨ Возможности
- Регестрация
- Создание/редактирование/удаление заметок
- Тегирование и поиск
- Избранное
- Локальное хранение (SQLite)

## 🛠️ Стек
- C# / .NET 10.0
- WPF + MVVM
- Entity Framework Core + SQLite

## 🚀 Быстрый старт
```bash
git clone https://github.com/your/NoteTaker
dotnet restore
dotnet ef database update
dotnet run
